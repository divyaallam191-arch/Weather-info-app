# Weather-info-app
A responsive weather application built with HTML,CSS and JAVASCRIPT that fetches real-time weather data based on the city entered by the user.

# Weather Application
A simple weather application built using HTML, CSS, and JavaScript.

## Features
- Search weather by city name
- Displays temperature
- Shows humidity
- Shows wind speed
- Responsive UI

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Weather API

## How to Run
1. Clone the repository
2. Open weather.html in a browser

## Future Improvements
- 5-day forecast
